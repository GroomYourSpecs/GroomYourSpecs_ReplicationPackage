# Controlled Experiment

## Installation

Using python3 and pip:

```sh
pip3 install notebook
pip3 install stats matplotlib scipy numpy==1.24 pandas seaborn plot_likert
jupyter notebook
```

## Overview
```
.
|---artefacts
   |---artefacts.zip: Artefacts produced during the controlled experiment
   |---artefacts_analysis.ipynb: Jupyter notebook with the data analysis
   |---bt_deliver_goods.csv: Measured target qualities for the deliver goods mission
   |---bt_food_logistics.csv: Measured target qualities for the food logistics mission
   |---goal_model.csv: Measured target qualities for the top-down strategy for both missions
   |---top_down_grountruth.zip:
   |---bottom_up_groundtruth.zip:
|---instructions
   |---bt_tutorial.pdf: Groot tutorial provided to experiment participants 
   |---goal_model_tutorial.pdf: MutRose tutorial provided to experiment participants
   |---missiona_food_logistics.pdf: Mission A in natural language description provided to experiment participants 
   |---missionb_deliver_goods_equipment.pdf: Mission B in natural language description provided to experiment participants
   |---schedule_lab1.pdf: Lab 1 schedule provided to experiment participants 
   |---schedule_lab2.pdf: Lab 2 schedule provided to experiment participants
|---questionnaire
   |---bottom_up.csv: Questionnaire responses for the bottom-up strategy
   |---top_down.csv: Questionnaire responses for the top-down strategy
   |---questionnaire.csv: Questionnaire responses for both strategies combined
   |---quest_analysis.ipynb: Jupyter notebook with the data analysis
   |---questions_controlled_exp_quest.pdf: The questions presented to participants
   |---likert_plot_quest.ipynb: R code used to generate the Likert graphs
```
